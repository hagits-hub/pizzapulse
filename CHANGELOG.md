**איפה לקבל מפתח:** console.anthropic.com → API Keys → Create Key

## 🚀 Deploy — GitHub Actions

**קובץ:** `.github/workflows/deploy.yml`

**⚠️ חשוב:** אין `cache: 'npm'` — גורם לבעיות build!

```yaml
- name: Setup Node
  uses: actions/setup-node@v4
  with:
    node-version: 20
    # אין cache: 'npm' !
```

**כתובת האתר:** `https://YOUR_USERNAME.github.io/pizzapulse-il`
